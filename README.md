# 🚀 Pathfinding Algorithm Visualizer

Interactive web application for visualizing graph pathfinding algorithms step by step.

The project helps simplify the understanding of graph traversal and shortest-path algorithms through animated grid-based simulations and interactive controls.

---

## ✨ Features

- 📍 Step-by-step visualization of pathfinding algorithms
- 🧱 Add and remove obstacles dynamically
- 🎯 Select start and destination nodes
- ⚡ Adjustable animation speed
- 📐 Dynamic grid resizing
- 🧠 Multiple pathfinding algorithms support
- 🔔 Toast notifications for state handling and user feedback
- 🎨 Interactive and responsive UI

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

---

## 📚 Algorithms Implemented

- Breadth-First Search (BFS)
- Depth-First Search (DFS)
- Dijkstra’s Algorithm

---

## 🎮 How It Works

1. Determine the size of the grid
2. Add obstacles/walls on the grid
3. Place the start and target nodes
4. Choose a pathfinding algorithm
5. Watch the algorithm explore the grid step by step
6. Adjust the animation speed as you want
